里仁家传承

可行性分析

里仁家传承电子出版工作室

一、项目概述

定位：以深度家族访谈为核心方法，采集家族历史文化记忆，通过电子出版形式（网站+小程序）交付家风、家训、家族故事的系统化传承成果。

核心理念："慎终追远，民德归厚"——每个家庭都值得拥有一部属于自己的电子家史。

二、市场分析

有利因素：

  • 老龄化窗口期。中国60岁以上人口超3亿，大量老人有讲述意愿但缺乏专业渠道。80-90后子女有孝心付费意愿，但不知道去哪里找专业服务。
  • 家族企业传承潮。中国第一代民营企业家集中进入交班期，高净值家族对"家族精神传承"的需求远超"财富传承"。
  • 文化政策红利。国家层面推动中华优秀传统文化传承发展，口述历史、家谱编修有政策支持。
  • "断亲"焦虑。年轻人普遍面临与家族关系疏离的困境，反而激发对"根"的渴望。
  • 市场已验证付费意愿。浙江某回忆录品牌产品线：入门级5888元→城市中产12888元→高端家族传记21888元→家族全景纪录片48888元→文化工程108888元起。

挑战因素：

  • 概念认知不足。大多数人不知道"口述历史+电子出版"可以做成商业服务。
  • 信任门槛高。涉及家族私密信息，客户需要建立深度信任才能开口。
  • 非刚需。属于情感消费，决策周期长，复购率低。

天花板可观，但获客成本极高，适合做精品小工作室模式而非规模化平台

三、市场定位建议

不做：传统家谱编修（已有大谱师等成熟SaaS）、简单回忆录撰写（竞争激烈）

要做："家风传承型"深度访谈 + 结构化电子出版

独特卖点：

  • 方法论差异化——基于12关卡框架设计结构化访谈体系
  • 产品形态差异化——不是纸质书或简单视频，而是网站+小程序的数字传承资产
  • 内容深度差异化——把家族故事放在时代大背景下，让个体记忆具有历史价值

命名建议：对外使用"里仁家传承"为主品牌，"电子出版"偏技术描述，客户感知不到价值。

四、目标客户分层

层级 客户画像 客单价 核心需求
L1 大众层 普通中产家庭，有老人健在 3,000-8,000元 为长辈做一份有仪式感的生命记录
L2 核心层 重视家风教育的家庭 8,000-30,000元 系统整理家族故事，形成家风档案
L3 高端层 家族企业主、文化名人 30,000-100,000元 家族精神传承+品牌化呈现

五、产品与服务设计

A. 家族记忆采集服务（收入主力）

  • 前期调研（1周）— 收集家族基本信息、照片、老物件清单
  • 访谈提纲定制（3天）— 根据12关卡框架设计个性化问题
  • 深度访谈（2-4次，每次2-3小时）— 每次聚焦一个主题
  • 素材整理与编辑（1-2周）— 文字转录、照片数字化、音频剪辑
  • 初稿交付（1周）— 电子家史初版
  • 修订确认（1周）— 家族成员共同审阅
  • 成品交付（3天）— 网站+小程序上线

B. 电子出版产品形态

  • 家族记忆网站 — 响应式设计，手机优先
  • 微信小程序 — 方便家族成员日常访问和分享
  • 数字纪念册 — PDF/EPUB格式，可打印
  • 短视频系列 — 每次访谈剪辑成3-5分钟短视频

C. 定价策略

套餐内容价格区间
入门版1位老人 × 2次访谈 + 基础网站5,888元
标准版3位成员 × 各2次访谈 + 网站+小程序19,800元
尊享版全家族多代访谈 + 完整电子出版 + 短视频49,800元
定制版按需协商100,000元+

六、营销策划

冷启动三步走：

  • 第一步：自己做第一个案例（免费/低价）——选择自己的家族（父母、祖辈）作为首个完整案例，制作作品集
  • 第二步：公众号内容种草（持续输出）——围绕"家风""家族记忆""口述历史"写文章
  • 第三步：口碑裂变——每个交付案例邀请客户写推荐语，老客户推荐新客户给佣金

关键营销话术：

"你家的故事，值得被记住"
"趁他们还在，把家族的记忆留下来"
"每个家庭都是一部微缩的中国史"

七、质量评估体系

访谈质量标准：

  • 真实性——不虚构、不美化，保留叙述者的原始表达风格
  • 完整性——覆盖人生关键节点（出生、成长、婚恋、事业、重大历史事件亲历）
  • 深度性——不止于事实陈述，要挖掘情感、价值观、人生智慧
  • 系统性——按12关卡框架组织，避免碎片化

交付物质量标准：

  • 文字稿——逐字转录+精编，保留口语特色但去除冗余
  • 多媒体——高清照片扫描、音频/视频剪辑达到广播级
  • 排版设计——手机优先、微信公众号风格（16px正文、2.0行高、卡片式布局）

八、客户关系管理

客户旅程：

认知 → 兴趣 → 咨询 → 签约 → 访谈 → 编辑 → 交付 → 分享 → 复购/转介绍

关键触点对策：

  • 首次接触——30分钟内响应，提供免费家族记忆评估（15分钟电话）
  • 签约前——发送案例作品集+FAQ文档，消除疑虑
  • 交付时——举办线上"家族记忆发布仪式"，邀请全体家族成员参加
  • 交付后——30天/90天/180天三次回访，推送家族纪念日提醒

长期关系维护：

  • 家族档案库 — 为每个客户建立独立档案
  • 年度更新提醒 — 每年生日、清明、中秋自动提醒
  • 家族社群 — 建立客户群，定期分享家族故事
  • 转介绍激励 — 老客户推荐新客户，双方各得一次免费访谈

十、风险评估与应对

风险概率影响应对措施
获客困难高高先用自家案例做标杆，公众号持续输出内容
交付周期长中中标准化流程，总周期不超过6周
客户期望过高中中签约前明确交付范围和标准
隐私泄露低极高签署保密协议，数据存储加密
个人精力瓶颈高中前12个月亲自做，积累后考虑招募兼职

十一、与"找到啦"项目的协同

这个项目与你已有的"找到啦"12关卡体系天然契合：

  • 方法论复用——12关卡框架可以直接转化为访谈提纲
  • 内容互通——家族访谈中的典型案例可以匿名化后用于找到啦的内容
  • 流量互导——找到啦的用户可以转化为里仁家传承的客户
  • 技术复用——网站和小程序的技术栈可以共享

建议：两个项目并行推进，资源分配比例7:3（找到啦为主，里仁家传承为辅）。

十二、结论与建议

总体评价：可行，但需谨慎

优势：市场需求真实存在且在增长；与你现有知识储备高度匹配；启动成本低，适合个人工作室模式；社会价值高。

劣势：市场教育成本高，获客困难；非标准化服务，难以规模化；个人精力有限。

行动建议（分阶段）

1
第一阶段（1-3个月）：验证期
用自家案例做第一个完整项目（免费），产出作品集，验证流程和方法论。公众号开始持续输出相关内容。
2
第二阶段（4-6个月）：试水期
接1-2个低价订单（成本价），积累真实案例。完善标准化流程文档，测试不同定价和套餐组合。
3
第三阶段（7-12个月）：正式运营
按正常定价接单，建立转介绍机制，评估是否值得全职投入。

底线原则

前6个月不要辞职全职做，保持副业身份验证商业模式。
如果12个月后月均收入能稳定超过1.5万，再考虑全职。
