# 案例：褚国祥-亲临现场验尸以真相息讼，不费公帑

**所属关卡：** Level08-人际迷宫
**来源：** 《智囊》

## 1. 人生场景关键词

亲临现场、快速核查、低成本息讼

## 2. 时代背景

明代湖州府司法流程繁琐，民间诉讼常因勘验拖延、吏员舞弊耗费大量公帑，民众也因讼事拖累生计。褚国祥任湖州添设贰守时，秉持宽平简易的治理风格。

## 3. 主要人物面临的问题和困境

- 发生妻子病亡被诬告为殴死的诉讼，按常规流程需发文书、遣吏员勘验，耗时耗财
- 吏员可能借机舞弊，反而激化矛盾
- 需快速查明真相，既安抚原告，又不浪费公帑

## 4. 主要人物采取的应对方式

### 积极应对
褚国祥收到诉状后故意不发文遣吏，下午突然带随从直奔原告家中，此时死者尚未入殓，当场验尸确认无殴打致死痕迹，仅薄责原告便了结此案，未花一分钱。

### 冯梦龙评语
"赵豫以缓，褚国祥以捷，其以安民为心一也。"

## 5. 事件的阶段成果及影响

- 短期：快速了结诬告诉讼，避免公帑浪费，原告心服口服
- 长期：褚国祥高效务实的作风获得民众信任，当地诬告之风得到遏制

## 6. 案例延伸的思考与可提炼的思维模型

### 延伸领域
基层执法、舆情核查、项目管理、问题排查

### 思维模型
- **现场核查模型**：跳过中间流程直奔现场，用第一手信息快速决策，避免信息失真和流程损耗
- **低成本解纷模型**：以最小资源投入查明真相，既解决问题又不造成额外负担

人生困境关键词：诬告案如何快速查明真相，通过亲临现场验尸，不费公帑高效息讼。
