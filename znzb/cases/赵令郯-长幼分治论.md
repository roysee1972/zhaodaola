# 案例：赵令郯-对不同对象用不同方法，长者用公法幼者用家教

**所属关卡：** Level08-人际迷宫
**来源：** 《智囊》

## 1. 人生场景关键词

分治策略、公私分明、因人制宜

## 2. 时代背景

北宋崇宁年间，朝廷在三京设置敦宗院安置宗室闲散人员，选拔宗子中的贤者管理。赵令郯被任命为南京敦宗院负责人，宗室成员身份特殊，治理难度大。

## 3. 主要人物面临的问题和困境

- 宗室成员身份尊贵，有的辈分高于管理者，难以用常规方式约束
- 年长宗室不服管教，年幼宗室需要教导而非惩罚
- 需找到既不伤宗室体面，又能有效治理的方法
- 人生困境关键词：管理身份特殊的群体难以约束，通过公法家法分治、因人制宜化解治理难题。

## 4. 主要人物采取的应对方式

### 积极应对
赵令郯向皇帝献策：辈分高于自己的宗室成员用国法约束，辈分低于自己的用家法教导。皇帝称赞其方法，任命他赴任。赵令郯到任后，宗室成员皆受教化，不再扰民，京邑得以安宁。

## 5. 事件的阶段成果及影响

- 短期：宗室治理有序，京邑安宁，赵令郯晋升
- 长期：分治策略成为后世管理特殊群体的典范，对不同对象用不同方法，效果显著

## 6. 案例延伸的思考与可提炼的思维模型

### 延伸领域
组织管理、家庭教育、特殊群体治理、差异化领导

### 思维模型
- **分治策略模型**：对不同群体采用不同的治理方式，扬长避短
- **公私分明模型**：公事用法，私事用情，公私兼顾，张弛有度
