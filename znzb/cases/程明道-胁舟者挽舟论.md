# 案例：程明道-胁舟者挽舟论

**所属关卡：** Level05-婚恋家庭
**来源：** 《智囊》卷三·迎刃

## 1. 人生场景关键词
化敌为业 / 絜矩之道 / 以工代罚

## 2. 时代背景
北宋时期，广济、蔡河沿岸有不逞之民专门胁取舟人钱物，每年焚烧舟船十余艘以立威，商旅深受其害，地方治安混乱。常规惩治只能抓现行，无法根治乱象，百姓和商旅苦不堪言。

## 3. 主要人物面临的问题和困境
- 濒河不逞之民专以胁取舟人钱物为业，每年焚舟十数立威，商旅不敢反抗，怨声载道
- 常规惩治只能抓现行，无法根治，抓了一批又冒出一批，陷入恶性循环
- 若加重刑罚，可能引发报复性焚舟，危害更大，治理难度高
- 这些人没有正当职业，靠勒索为生，难以通过常规手段引导转型

## 4. 主要人物采取的应对方式（含原文引用与冯梦龙评语）

#### 积极应对
1. 捕得一人，令其引荐同党，原文引用："明道至，捕得一人，使引其类，得数十人"，一次性掌握整个胁舟群体。
2. 不追究过往恶行，给改过机会，原文引用："不复根治旧恶"，避免激起群体对立。
3. 分地安置，令其以挽舟为业，同时负责监察为恶者，原文引用："分地而处之，使以挽舟为业，且察为恶者"，将胁迫者转化为合法的挽舟从业者，实现自我监管。
4. 彻底解决焚舟问题，原文引用："自是境无焚舟之患"，治安得到根本好转。

#### 消极应对
无（程颢全程主动施策，化恶为善，无消极行为）

冯梦龙评："胁舟者业挽舟，使之悟絜矩之道，此大程先生所以为真道学也！"

## 5. 事件的阶段成果及影响
- **短期效果**：自是境无焚舟之患，商旅安全得到保障，地方秩序迅速恢复。
- **长期影响**：将黑产从业者转化为正规挽舟从业者，既解决了治安问题，又给了他们正当生路，实现长治久安。
- **社会影响**：成为"化恶为善"的治理典范，冯梦龙高度评价程颢的"真道学"实践，体现了儒家治理的现实价值。

## 6. 案例延伸的思考与可提炼的思维模型

#### 延伸领域
黑产治理、社区矫正、失业人群就业引导、行业乱象整治、刑满释放人员安置

#### 思维模型
- **胁舟挽舟模型**：将违规群体转化为正规行业从业者，用合法职业替代非法生计，从根源消除问题动机。
- **絜矩之道模型**：让作恶者体验正当职业的价值，领悟推己及人的道理，实现自发行为改变。
- **以工代罚模型**：不追究过往恶行，给足改过机会，用正当职业替代非法收入来源，降低再犯率。

人生困境关键词：如何根治黑产乱象，通过化敌为业、以工代罚实现长治久安。
