# 案例：裴晋公-丢印从容论

**所属关卡：** Level04-独立谋生
**来源：** 《智囊》卷三·迎刃

## 1. 人生场景关键词
临危不乱 / 智量宏大 / 从容破局

## 2. 时代背景
唐代中期，裴度任中书省长官，中枢政务繁复，胥吏作弊现象时有发生。官印是行政核心凭证，丢失官印属于重大政务事故，若处理不当会引发行政混乱，甚至被别有用心之人利用，造成难以挽回的损失。

## 3. 主要人物面临的问题和困境
- 左右随从突然报告官印丢失，属于重大政务事故，常规处理会急捕胥吏、大动干戈
- 若急于追查，胥吏可能将官印投入水火毁灭，彻底无法找回
- 事故发生在宴饮场合，若慌乱会导致在场官员士绅恐慌，引发不必要的舆情
- 作为中枢长官，需在短时间内做出正确决策，避免事态扩大影响朝政

人生困境关键词：官印丢失属重大事故，如何在不惊动众人的情况下从容找回，缓之则复还急之则毁。

## 4. 主要人物采取的应对方式（含原文引用与冯梦龙评语）

#### 积极应对
1. 得知印失后怡然自若，严禁左右声张，原文引用："公怡然，戒勿言"，避免引发恐慌和胥吏的过激销毁行为。
2. 照常张宴举乐，不耽误正常公务社交，原文引用："方张宴举乐，人不晓其故"，稳住在场人员情绪，不暴露事故。
3. 半夜宴酣时，左右再报印已找回，裴度仍不答，宴饮极欢而罢，原文引用："夜半宴酣，左右复白印存，公亦不答，极欢而罢"。
4. 事后解释原因："胥吏辈盗印书券，缓之则复还故处，急之则投水火，不可复得矣！"，精准判断胥吏盗印的动机是临时使用，而非恶意销毁。

#### 消极应对
无（裴度全程冷静从容，无任何慌乱或激进处置）

冯梦龙评："不是矫情镇物，真是透顶光明，故曰'智量'，智不足，量不大。"

## 5. 事件的阶段成果及影响
- **短期效果**：官印完好找回，未造成任何政务损失，宴饮正常结束，在场人员无一知晓事故。
- **长期影响**：展现了裴度过人的智量和从容气度，巩固了其中枢执政的权威和口碑。
- **社会影响**：成为"智量"的典范案例，冯梦龙专门点评，强调"智不足，量不大"的核心道理，被后世官员奉为处事圭臬。

## 6. 案例延伸的思考与可提炼的思维模型

#### 延伸领域
职场重大危机处理、高管情绪管理、行政事故调查、团队突发问题应对、核心资产丢失处置

#### 思维模型
- **缓急得宜模型**：遇到重大突发事故时，不急于行动，先判断涉事方的行为逻辑，用时间换空间，避免激化矛盾。
- **智量从容模型**：越是重大危机越保持情绪稳定，避免慌乱决策导致事态扩大，从容本身就是一种威慑力。
- **盗印归还原位模型**：针对内部人员的小过错，给足台阶和缓冲时间，让其主动纠正错误，而非穷追猛打。
