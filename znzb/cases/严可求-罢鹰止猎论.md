# 案例：严可求-罢鹰止猎论

**所属关卡：** Level04-独立谋生
**来源：** 《智囊》卷三·迎刃

## 1. 人生场景关键词
断其利源 / 禁于未然 / 简政息争

## 2. 时代背景
五代十国时期，徐温（烈祖）辅佐吴国，四方战事频繁，军校常聚集在郊野打猎饮乐，有时会骚扰百姓，影响地方秩序。烈祖想用法令纠正，但又需要依靠军校的武力平定四方，陷入"要军心还是要民生"的两难境地。

## 3. 主要人物面临的问题和困境
- 军校多在郊野打猎聚饮，骚扰百姓，影响地方安定，民怨渐起
- 烈祖想用法令约束，但当前需要依靠军校的武力平定四方，不宜直接处罚
- 明令禁止可能引发军校不满，影响军心，干扰战事推进
- 常规治理手段（惩戒、禁令）均不适用，直接行动会顾此失彼

人生困境关键词：军校骚扰百姓但不可明令禁止，需断其利源间接治理，不令而止。

## 4. 主要人物采取的应对方式（含原文引用与冯梦龙评语）

#### 积极应对
1. 劝阻烈祖直接绳之以法，原文引用："无烦绳之，易绝耳"，提出更柔性的解决方案。
2. 建议切断军校打猎的必要条件，原文引用："请敕泰兴、海盐诸县，罢采鹰鹯"，停止供应猎鹰，让军校无猎可打。
3. 不用明令禁止，自然消除军校游猎行为，原文引用："可不令而止"，避免激化军校与朝廷的矛盾。

#### 核心启示
断其利益来源，不必明令禁止，问题自然消解。

#### 消极应对
无（严可求全程提供间接治理方案，无消极行为）

## 5. 事件的阶段成果及影响
- **短期效果**：期月之间，禁校无复游墟落者，军校不再骚扰百姓，地方秩序恢复。
- **长期影响**：既维护了地方民生，又没有得罪军校群体，保障了吴国军事力量的稳定，为平定四方提供了支持。
- **社会影响**：成为"不令而止"、间接治理的典型案例，避免了直接冲突，为后世处理类似两难问题提供了参考。

## 6. 案例延伸的思考与可提炼的思维模型

#### 延伸领域
团队管理（减少无效行为）、行业监管（断利益链）、青少年行为引导、企业成本控制、冲突柔性化解

#### 思维模型
- **罢鹰止猎模型**：要禁止某种行为，不必明令约束，直接切断其利益来源或必要条件，问题自然消解。
- **不令而止模型**：针对依赖特定资源的群体，通过取消资源供应替代直接禁令，避免激化矛盾。
- **两难平衡模型**：在需要兼顾两方利益（如军权与民生）时，用间接手段达成目标，不走非此即彼的极端。
