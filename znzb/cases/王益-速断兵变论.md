# 案例：王益-果断处首恶，以速度化被动为主动

**所属关卡：** Level08-人际迷宫
**来源：** 《智囊》

## 1. 人生场景关键词

快速决策、处首震众、兵变平息

## 2. 时代背景

宋代韶州屯兵五百人，因换防延迟企图兵变，事情泄露后全郡震惊。王益任韶州知州，需快速平息叛乱，避免事态扩大。

## 3. 主要人物面临的问题和困境

- 屯兵谋划兵变，消息泄露后人心惶惶
- 若按常规流程将首恶下狱，可能引发夜劫监狱的二次叛乱
- 需快速处置，震慑余党，避免兵变发生
人生困境关键词：兵变一触即发，如何当日速断首恶不下狱，以果断处置震慑余众止乱。

## 4. 主要人物采取的应对方式

### 积极应对
王益当场逮捕五名首恶，当日就判处流刑，拒绝将他们下狱。其余叛兵听说若下狱会被夜劫，反而对王益的果断处置心服口服。

## 5. 事件的阶段成果及影响

- 短期：兵变未爆发，首恶被流放，韶州恢复安定
- 长期：王益的果断作风震慑了驻军，当地再未发生类似兵变

## 6. 案例延伸的思考与可提炼的思维模型

### 延伸领域
危机处理、团队管理、突发事件应对、舆情管控

### 思维模型
- **速断止乱模型**：对突发事件的核心责任人快速处置，切断混乱源头
- **震慑余众模型**：对首恶从重从快处理，让其余参与者放弃抵抗念头
