# 案例：王旦-以势压人，不战而屈人之兵

**所属关卡：** Level08-人际迷宫
**来源：** 《智囊》

## 1. 人生场景关键词

外交威慑、不战而胜、心理战

## 2. 时代背景

宋真宗时期，西夏李德明（李元昊之父）向宋朝索要粮食万斛，意图试探宋朝实力，同时获取实际利益。王旦任宰相，需在不引发战争的前提下，让西夏主动放弃无理要求。

## 3. 主要人物面临的问题和困境

- 西夏索要万斛粮食，直接拒绝可能引发边境冲突，给予又会助长其贪欲
- 需展示宋朝实力，让西夏知难而退
- 要避免战争，同时维护宋朝权威

人生困境关键词：如何以超出预期的实力展示回应试探，不战而屈人之兵维护权威。

## 4. 主要人物采取的应对方式

### 积极应对
王旦下令让有关部门在京城准备百万石粮食，下诏让赵德明亲自来取。赵德明得知后十分羞愧，感叹"朝廷有人"，随即放弃索粮要求。

## 5. 事件的阶段成果及影响

- 短期：西夏主动放弃索粮，边境避免冲突
- 长期：西夏认识到宋朝实力雄厚，不敢轻易挑衅，边境维持长期稳定

## 6. 案例延伸的思考与可提炼的思维模型

### 延伸领域
商务谈判、竞争博弈、危机威慑、边界维护

### 思维模型
- **不战屈兵模型**：展示远超对方预期的综合实力，让对方主动放弃不合理要求
- **心理威慑模型**：用超出对方承受能力的方案回应试探，无需实际付出即可达成目标
