# 案例：龚遂-形貌短小治理渤海论

**所属关卡：** Level03-初入人海
**来源：** 《智囊》卷三·迎刃

## 1. 人生场景关键词
临危受命 / 简政安民 / 便宜行事

## 2. 时代背景
西汉宣帝时期，渤海郡及周边地区连年饥荒，百姓生存无着，盗贼四起，地方二千石官员无力遏制乱象。当时地方治理多依赖严苛文法，官吏苛待百姓，官民对立严重，朝廷对渤海乱局束手无策，急需能稳定地方的官员赴任。

## 3. 主要人物面临的问题和困境
- 年已七十，形貌短小，初任即被宣帝轻视，面临朝廷对其能力的质疑
- 渤海饥荒连年，盗贼群起，前任长官无法遏制乱象，治理基础极差
- 百姓困于饥寒，官吏不恤民情，官民矛盾已激化到百姓"盗弄陛下之兵"的程度
- 朝廷期望快速平乱，但单纯动武可能进一步激化矛盾，引发更大动荡

人生困境关键词：如何在饥荒盗贼四起的乱局中安民而非激化矛盾，通过争取便宜行事权以安抚代征剿。

## 4. 主要人物采取的应对方式（含原文引用与冯梦龙评语）

#### 积极应对
1. 面见宣帝时先剖析乱源，原文引用："海濒辽远，不沾圣化，其民困于饥寒而吏不恤，故使陛下赤子盗弄陛下之兵于潢池中耳"，精准指出问题根源是吏治失当而非百姓本恶。
2. 反问宣帝治理意图，原文引用："今欲使臣胜之耶，将安之也？"，引导皇帝明确"安民"的核心方向，避免被要求用武力平乱。
3. 主动争取施政自主权，原文引用："无拘臣以文法，得一切便宜从事"，获得灵活施政的授权，避免被僵化规则束缚。
4. 到任渤海后精简行政：遣还迎候之兵，罢逐捕盗贼之吏，明确界定"凡持兵器者乃为盗贼，其余持钩田器者皆为良民"，分化盗贼群体。
5. 单车独行至府，示以无诛杀之意，让盗贼主动解散，原文记载："盗贼闻教令，即时解散，弃兵器而持钩"。

#### 消极应对
无（龚遂全程主动施策，无消极应对行为）

冯梦龙评："古之良吏，化有事为无事，化大事为小事，蕲于为朝廷安民而已。今则不然，无事弄做有事。"

## 5. 事件的阶段成果及影响
- **短期效果**：到任后未动干戈，盗贼即时解散，渤海郡迅速恢复秩序，百姓重归农业生产。
- **长期影响**：稳定了渤海地区的民生，验证了"安民优于胜之"的治理思路，龚遂也因此获得宣帝信任。
- **社会影响**：成为古代良吏治理的典范，其"便宜从事""良盗区分"的策略被后世地方官推崇效仿。

## 6. 案例延伸的思考与可提炼的思维模型

#### 延伸领域
职场新人接手问题团队、基层危机管理、地方治理、群体事件处置、职场向上争取授权

#### 思维模型
- **便宜行事模型**：向上级明确目标后，争取灵活施政的自主权，避免僵化规则束缚，根据实际场景调整策略。
- **良盗区分模型**：处理群体性问题时，精准区分核心矛盾群体与被动参与者，避免扩大打击面，降低治理成本。
- **示诚消解模型**：以退为进，通过展示无诛杀、无压迫的诚意，消解对立群体的抵抗意愿，从根源解决问题。
