# 案例：马默-修正制度的残酷，体现人道关怀

**所属关卡：** Level08-人际迷宫
**来源：** 《智囊》

## 1. 人生场景关键词

制度修正、人道关怀、废除酷刑

## 2. 时代背景

宋代沙门岛是流放罪犯之地，定额三百人，超员后便将罪犯投入海中杀死。寨主李庆一任期内杀了七百多人，极其残忍。马默任登州知州，上书朝廷改革这一残酷制度。

## 3. 主要人物面临的问题和困境

- 旧制度极其残忍，超员即杀人，违背人道的治国理念
- 直接废除会遇到既得利益者的反对
- 需找到既不引发监狱爆满，又让制度更人道的解决方案

人生困境关键词：如何改革延续已久的残忍制度，通过合理建议兼顾人道与秩序，实现制度改良。

## 4. 主要人物采取的应对方式

### 积极应对
马默上书建议：超员时，挑选年深、在配所无过犯的罪犯移至登州。神宗采纳，著为定制，挽救了许多人的性命。

## 5. 事件的阶段成果及影响

- 短期：许多罪犯得以存活，制度得到人性化改革
- 长期：沙门岛虐囚现象彻底杜绝，马默的仁政被传颂，后世子孙满堂

## 6. 案例延伸的思考与可提炼的思维模型

### 延伸领域
制度建设、法律改革、人道主义、公共政策

### 思维模型
- **制度修正模型**：发现制度缺陷后，通过合理建议推动改革，而非强行废除
- **人道关怀模型**：在制度框架内注入人道精神，让冰冷规则变得有温度
