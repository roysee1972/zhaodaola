# 案例：张忠献-鸿门宴式布局，以诏书之名瓦解军阀势力

**所属关卡：** Level08-人际迷宫
**来源：** 《智囊》

## 1. 人生场景关键词

鸿门宴、调虎离山、兵不血刃

## 2. 时代背景

南宋初年，叛将范琼拥兵割据长江上游，召他不来，来了又不肯交兵权，朝野震惊。张忠献（张浚）任宰相，需除掉范琼但避免引发兵变。

## 3. 主要人物面临的问题和困境

- 范琼拥兵自重，公然抗命，威胁朝廷安全
- 直接派兵征剿会引发大规模战乱，伤亡惨重
- 需设计让范琼主动交权，同时安抚其部下

人生困境关键词：如何兵不血刃地除掉拥兵自重的叛将，通过设宴诱捕分化其部下。

## 4. 主要人物采取的应对方式

### 积极应对
张浚先派张俊带千兵渡江假装捕盗，随后召范琼、张俊、刘光世到都堂议事，设宴款待。宴后，刘子羽突然拿出诏书，宣布范琼罪状，命他到大理寺对质。范琼惊愕失措，被当场逮捕，刘光世随即安抚范琼部下，告知只诛范琼一人，众人归顺，叛乱顷刻平定。

## 5. 事件的阶段成果及影响

- 短期：兵不血刃擒杀范琼，其部下归顺朝廷，避免大规模战乱
- 长期：稳定了南宋初年的局势，张浚的谋略成为后世处理军阀的典范

## 6. 案例延伸的思考与可提炼的思维模型

### 延伸领域
危机处理、商业并购、团队重组、反恐行动

### 思维模型
- **鸿门宴模型**：请君入瓮后突然发难，以权威名义瓦解对方抵抗意志
- **分而治之模型**：只惩处核心人物，安抚其余部众，快速稳定局面
