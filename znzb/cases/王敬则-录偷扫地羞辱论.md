# 案例：王敬则-录偷扫地羞辱论

**所属关卡：** Level05-婚恋家庭
**来源：** 《智囊》卷三·迎刃

## 1. 人生场景关键词
以盗治盗 / 羞辱教化 / 邻里监督

## 2. 时代背景
南齐时期，吴兴郡盗掠之风盛行，百姓财产和人身安全受到严重威胁，地方治安混乱。官府常规缉捕效果不佳，盗贼屡禁不止，抓了一批又来一批，治理陷入僵局，百姓苦不堪言。

## 3. 主要人物面临的问题和困境
- 郡中旧多剽掠，盗贼成群结队作案，常规治安手段无法遏制
- 缉捕成本高，抓了现行盗贼又会冒出新的，无法根治乱象
- 盗贼熟悉官府缉捕套路，流动性强，难以彻底清剿
- 若加重刑罚，可能引发盗贼更激烈的对抗，反而危害百姓
人生困境关键词：盗贼屡禁不止常规手段失效时，通过以盗治盗、亲属施压和羞辱教化实现低成本治理。

## 4. 主要人物采取的应对方式（含原文引用与冯梦龙评语）

#### 积极应对
1. 抓获一个盗贼后，召集其亲属到场，鞭打数十下，原文引用："敬则录得一偷，召其亲属于前，鞭之数十"，让亲属知晓其恶行，利用亲属压力约束盗贼行为。
2. 命令盗贼长期负责扫街，原文引用："使之长扫街路，久之乃令举旧偷自代"，既作为惩罚，又给出改过机会，同时利用盗贼熟悉同类群体的特点。
3. 待盗贼熟悉情况后，令其举荐旧偷自代，用盗贼管理盗贼，原文："诸偷恐为所识，皆逃走，境内以清"，利用盗贼之间的互相监督和恐惧心理，不费多力清除盗贼。

#### 消极应对
无（王敬则全程主动创新治理手段，无消极行为）

冯梦龙评："辱及亲属，亲属亦不能容偷矣。唯偷知偷，举偷自代，胜用缉捕人多多矣！"

## 5. 事件的阶段成果及影响
- **短期效果**：诸偷恐被识破举荐，全部逃走，吴兴郡境内迅速恢复清净，治安大幅好转。
- **长期影响**：形成了"以盗治盗"的有效治理模式，大幅降低了地方缉捕成本，为后世治安治理提供了新思路。
- **社会影响**：成为古代治安治理的创新典型案例，冯梦龙高度评价，认为这种模式比常规缉捕手段高效数倍。

## 6. 案例延伸的思考与可提炼的思维模型

#### 延伸领域
社区治理、团队内部监督、青少年犯罪矫正、员工舞弊防范、流动人口管理

#### 思维模型
- **录偷扫地模型**：抓获违规者后，结合公开羞辱、亲属施压、长期劳动改造和社会监督，实现低成本行为矫正。
- **以盗治盗模型**：利用熟悉群体规则的内部人去管理同类群体，比外部监管更精准高效，大幅降低管理成本。
- **亲属连坐教化模型**：将个人过错与亲属声誉绑定，利用亲属的压力约束个体行为，避免单纯靠惩罚激化矛盾。
