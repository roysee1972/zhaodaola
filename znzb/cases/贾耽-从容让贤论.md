# 案例：贾耽-面对权力交接大度从容，全身而退

**所属关卡：** Level08-人际迷宫
**来源：** 《智囊》

## 1. 人生场景关键词

权力交接、从容退让、全身而退

## 2. 时代背景

唐代节度使贾耽任山南东道节度使时，朝廷突然下诏让行军司马樊泽接替他的职位。贾耽需平稳完成权力交接，避免部将哗变。

## 3. 主要人物面临的问题和困境

- 突然被撤换，部将不满，可能发动兵变
- 若滞留不走会引发朝廷猜忌，若强行压制部将会导致内乱
- 需平稳交接，保全自身，同时稳定军府
人生困境关键词：突然被撤换部将不满，如何从容退让带刺同行平稳完成权力交接。

## 4. 主要人物采取的应对方式

### 积极应对
贾耽收到诏书后藏入怀中，宴席上神色不变。宴罢后立即命令将吏拜谒樊泽。部将张献甫想要杀樊泽，贾耽以天子任命为由劝阻，当日就离开军镇，带张献甫随行，军府迅速安定。

## 5. 事件的阶段成果及影响

- 短期：平稳完成权力交接，未引发兵变，军府安定
- 长期：贾耽的从容大度成为唐代节度使交接的典范，他本人也得以善终

## 6. 案例延伸的思考与可提炼的思维模型

### 延伸领域
职场离职、团队交接、权力过渡、高管任免

### 思维模型
- **从容退让模型**：面对权力变动保持冷静，主动配合交接，避免无谓冲突
- **带刺同行模型**：将不满的核心下属带离原单位，消除稳定隐患
