# 案例：张忠定-从细节体察人情，以仁心化人心

**所属关卡：** Level12-终篇传承
**来源：** 《智囊》

## 1. 人生场景关键词

细节体察、仁心管理、下属关怀、人心归附

## 2. 时代背景

北宋张忠定公（张咏）任地方官时，退堂后看到厅子（差役）熟睡，没有像常人那样责罚，而是细心询问缘由，发现差役母亲久病、兄长出外未归，因心中烦闷才在当值时不小心睡着。

## 3. 主要人物面临的问题和困境

- 差役当值熟睡，按规矩应当责罚
- 若直接责罚，会激化差役的家庭困境，引发更大问题
- 需要体察下属的真实处境，而不是机械执行规则

人生困境关键词：面对下属违规，如何体察背后隐情以仁心代替责罚凝聚人心。

## 4. 主要人物采取的应对方式（含原文引用与冯梦龙评语）

#### 积极应对
张忠定询问差役家事，确认情况后，派场务一名给差役，照顾其家庭。原文引用：*"公诘之：'汝家有甚事？'对曰：'母久病，兄为客未归。'访之果然。公翌日差场务一名给之，且曰：'吾厅岂有敢睡者耶？此必心极幽懑使之然耳，故悯之。'"*

#### 冯梦龙评语
*"体悉人情至此，人谁不愿为之死乎！"*

## 5. 事件的阶段成果及影响

- 短期：差役得到帮助，家庭困境缓解，更加忠心效命
- 长期：张忠定的仁名传开，下属都愿意为他效死力
- 管理影响：成为"仁者无敌"的管理典范，说明体察人情比严苛规则更能凝聚人心

## 6. 案例延伸的思考与可提炼的思维模型

#### 延伸领域
现代企业管理、团队领导、基层公务员管理、HR员工关怀

#### 思维模型
- **细节体察模型**：从下属的异常行为入手，挖掘背后的真实原因，而非直接处罚
- **仁心凝聚模型**：对下属的关怀比规则威慑更能获得真心归附
