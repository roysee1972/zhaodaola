# 案例：吴时来-多谋善断，文武双全的综合治理

**所属关卡：** Level08-人际迷宫
**来源：** 《智囊》

## 1. 人生场景关键词

综合治理、文武双全、安民御寇

## 2. 时代背景

明代嘉靖年间倭寇肆虐，各地援兵抵达松江，因犒赏不及时喧哗鼓噪。吴时来任推官，需安抚援兵、抵御倭寇、安置流民，局势错综复杂。

## 3. 主要人物面临的问题和困境

- 援兵因犒赏延迟喧哗，处置不当会引发兵变
- 客兵彪悍，剽掠百姓，与倭寇无异
- 流民涌入，闭城会引发不满，开城会拥挤生疫
- 城墙崩塌，倭寇来袭，兵力不足
人生困境关键词：如何同时应对兵变、流民、倭寇等多重危机，通过综合统筹、亲自垂范逐一化解。

## 4. 主要人物采取的应对方式

### 积极应对
1. 安置援兵：选福田禅林立营，按次序犒赏，公平高效，诸营安定。
2. 约束客兵：结好客兵首领，处治违纪者，客兵不敢犯民。
3. 安置流民：开城让流民择地居住，开四水关让运薪船载污物出城，防止瘟疫。
4. 抵御倭寇：城墙崩塌时仅用强弩扼守，断言泥泞倭寇无法登城；亲自带头出城运木建栅，三日修完城墙，倭寇北逃；决震泽水断敌路，斩首三千余。

### 延伸内容
寇势逼甚，士女趋保于城者万计。或议闭关拒之，时来悉纵人择闲旷地舍之。又城隘民众遂污蒸而为疫，时来乃四启水关，使输薪谷者因其归舟载秽滞以出。明年四月寇猝至攻城，雨甚城崩西南隅十余丈，人情汹汹。时来尽撤屯戍，第以强弩数十扼其冲，曰："淖泞，彼安能登？"果无恙。时内徙之民薄城而居，类以苫盖。时来虑为火箭所及，亟撤之而阴识其姓名于屋材，夜选卒运之城外以为木栅，捍修城者。卒皆股栗不前，时来首驰一骑出南门，众皆从之，平明栅毕，三日而城完。贼知有备北走，时来建议决震泽水断松陵道，贼至平望阻水不得进，我兵尾而击之，斩首三千余，溺死无算。

### 冯梦龙评语
"此公文武全才，故备载之。"

## 5. 事件的阶段成果及影响

- 短期：援兵安定，客兵不扰民，流民妥善安置，倭寇败退，松江保全
- 长期：吴时来的综合治理模式成为明代抗倭的典范，其文武全才被载入史册

## 6. 案例延伸的思考与可提炼的思维模型

### 延伸领域
危机管理、应急管理、城市治理、防汛抗台

### 思维模型
- **综合统筹模型**：同时处理多维度危机，分阶段优先级处置，避免顾此失彼
- **亲自垂范模型**：带头行动消除下属畏惧，快速完成急难险重任务
- **以患为利模型**：利用现有条件（如水关、震泽）转化为防御优势
