# 案例：张方平-在大国博弈中保持平衡，不轻易选边站

**所属关卡：** Level08-人际迷宫
**来源：** 《智囊》

## 1. 人生场景关键词

外交平衡、多方博弈、不选边站

## 2. 时代背景

北宋时期，西夏元昊称臣后与契丹产生矛盾，请求北宋断绝与契丹的往来。张方平任知谏院，需在宋、夏、辽三国博弈中做出最优选择。

## 3. 主要人物面临的问题和困境

- 西夏新近称臣，但契丹是北宋久和的强敌
- 若答应西夏要求，可能失去契丹的友好关系，引发边境危机
- 需在两大势力间保持平衡，不偏不倚
- 人生困境关键词：如何在多方博弈中不选边站，通过将矛盾踢回对方保持平衡。

## 4. 主要人物采取的应对方式

### 积极应对
张方平建议：赐诏给元昊让他自己处理与契丹的矛盾，待嫌隙消除后立即封册。这样西北两边都能保全。

## 5. 事件的阶段成果及影响

- 短期：北宋未被迫选边站，避免了外交困境
- 长期：张方平的建议让北宋在三国博弈中保持灵活，不失强敌

## 6. 案例延伸的思考与可提炼的思维模型

### 延伸领域
国际关系、商业竞争、多方谈判、战略选择

### 思维模型
- **平衡外交模型**：在大国博弈中保持中立，避免过早选边站
- **条件满足模型**：将球踢回给对方，让对方自己解决矛盾，己方坐收渔利
