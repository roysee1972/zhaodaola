# 案例：王琼-调虎离山，用赏赐之名行分兵之实

**所属关卡：** Level08-人际迷宫
**来源：** 《智囊》

## 1. 人生场景关键词

调虎离山、分化瓦解、兵不血刃

## 2. 时代背景

明武宗南巡归来弥留之际，杨廷和已定计擒拿权宦江彬。江彬统领数千精锐边兵，是其爪牙，杨廷和担心仓促动手会引发兵变，于是请教王琼。

## 3. 主要人物面临的问题和困境

- 江彬拥有数千精锐边兵，直接擒拿会引发兵变
- 江彬本人武力高强，单独动手风险极大
- 需先将边兵调离，再擒江彬
人生困境关键词：权臣拥有精锐兵力时如何擒拿，通过调虎离山以论功行赏名义分散其力量。

## 4. 主要人物采取的应对方式

### 积极应对
王琼建议：以记录扈从南巡之功的名义，让边兵到通州接受赏赐。边兵全部离开后，江彬势单力薄，被顺利擒获。

## 5. 事件的阶段成果及影响

- 短期：边兵被调离，江彬被擒获，未引发任何兵变
- 长期：权宦江彬伏诛，明武宗驾崩后朝廷得以平稳过渡，王琼的计策成为后世处理权臣的典范

## 6. 案例延伸的思考与可提炼的思维模型

### 延伸领域
危机处理、权力交接、反腐行动、商业竞争

### 思维模型
- **调虎离山模型**：将对方的核心支持力量调离，使其孤立无援
- **诱赏分兵模型**：以论功行赏的名义将对方爪牙调开，削弱其力量
